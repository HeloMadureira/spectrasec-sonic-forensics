# An-lise-forense-de-dados-por-meio-de-sonifica-o-e-processamento-de-sinais
Projeto que utiliza sonificação de dados e análise espectral para identificar criptografia, ofuscação e anomalias em arquivos binários. A abordagem permite “ouvir” a entropia dos dados como apoio à análise forense e segurança cibernética.
